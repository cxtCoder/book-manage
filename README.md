# book-manage
本工程是文史资料管理系统，分为管理端（book-console）和服务端（book-manage） 
+ book-console 
管理端：基于 ElementUI + VUE

+ book-manage
服务端：基于 SpringCloud + Eureka + Mysql

## 子模块说明
***
book-manage-api
API层子模块
***
book-manage-client
Controller层子模块
***
book-manage-server
Service层子模块
***
book-manage-database
数据源-Mybatis子模块
***
eureka
注册中心子模块
